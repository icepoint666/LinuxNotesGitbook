# 更改身份：su与sudo的区别

很多时候拥有另一个用户的身份是很重要

* 变为一个超级用户来执行一些管理任务
* 变成另一个普通用户来执行一些任务

三种方法

* 注销系统切换用户
* su命令
* sudo命令

**su命令**

以另一个用户和身份来启动shell

su \[user\] 不加user默认为管理员

切换为管理员执行命令

```bash
$ su -c 'ls -l /root/*'
# 类似于sudo ls -l /root/*
```

**sudo命令**

**类似于su，区别在于管理员可以通过配置sudo命令，允许普通用户以不同身份（一般是超级用户）来执行命令**
