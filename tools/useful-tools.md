# Useful Tools

#### xdg-open

一个linux命令：opens a file or URL in the user's preferred application

If a URL is provided the URL will be opened in the user's preferred web browser.

**shellcheck 静态分析**

应用于 shell 脚本的静态分析工具

检查运行脚本的一些warning，可能错误的地方

```text
$ shellcheck mcd.sh
```

**man**

查看命令的manual help

```bash
$ man mv
$ man convert
```
